# Progetti-Database

Questo repository raccoglie i progetti di tre diverse tipologie di Sistemi di Gestione di Basi di Dati (DBMS). Ogni cartella contiene una descrizione teorica generale e un'analisi completa. 

### Tipologie di Database Contenute:

* **[01. Database Relazionali](./01-database-relazionali)**
  * *Descrizione:* Strutture dati basate sul modello relazionale (tabelle, righe e colonne) con vincoli di integrità e utilizzo del linguaggio SQL per le query.
  * *Tecnologie di riferimento:* SQLite.

* **[02. Database NoSQL a Grafo](./02-database-a-grafo)**
  * *Descrizione:* Modello NoSQL focalizzato sulle relazioni tra i dati. Utilizza nodi (entità), archi (relazioni) e proprietà per rappresentare e mappare interconnessioni complesse in modo altamente efficiente.
  * *Tecnologie di riferimento:* Neo4j.

* **[03. Database NoSQL Key-Document](./03-database-key-document)**
  * *Descrizione:* Analisi dei modelli NoSQL orientati al Documento (memorizzazione in JSON) e Chiave-Valore (coppie di chiavi univoche associate a un valore), ideali per scalabilità e flessibilità del flusso di dati.
  * *Tecnologie di riferimento:* Elasticsearch.

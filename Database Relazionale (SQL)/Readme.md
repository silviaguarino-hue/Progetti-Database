# Database Relazionale (SQL) - Palestra Privata
In questa sezione è presente il materiale relativo allo studio e alla progettazione di un database relazionale di una palestra privata.

### Contenuti nella cartella
* **README.md**: Descrizione generale del progetto.
* **Analisi_Completa.pdf**: Analisi dei requisiti, schema relazionale (ER), schema Logico e query SQL.

### Panoramica del progetto
Il database è stato progettato per organizzare e automatizzare le attività quotidiane di una palestra privata. In particolare permette di:
* **Gestire le iscrizioni ai corsi**: Monitorare la partecipazione degli utenti, consentendo di estrarre in ogni momento l'elenco completo di tutti gli iscritti a uno specifico corso.
* **Pianificare il lavoro degli istruttori**: Tracciare le attività del personale ed eseguire report per mostrare quanti corsi tiene ciascun istruttore.
